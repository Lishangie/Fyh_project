ГОСТ-ориентированные правила форматирования таблиц

- Шрифт: Times New Roman, 14pt
- Выравнивание заголовков по центру, тела таблицы — по центру/лево в зависимости от типа данных
- Нумерация таблиц: Таблица 1 — название сверху, подпись снизу (по ГОСТ)
- Для динамических таблиц использовать Jinja2-теги `{% for row in table %}` и `%tr/%tc` шаблоны docxtpl

Требуйте точного соответствия числовых полей (без экспоненциальной нотации) и единиц измерения.
